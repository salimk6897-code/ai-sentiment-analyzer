# 🧠 AI Sentiment Analyzer - by salim malik

This was the project where things started feeling really exciting 
for me. I mean — I type a sentence, and AI tells me the emotion 
behind it? That's just cool! 

I built this while learning how to connect Python with Google's 
Gemini AI API. The idea was to send any sentence to the AI and 
get back structured data about the sentiment and emotion.

## What it does

You type any sentence and the program sends it to Google Gemini AI.
The AI analyzes it and returns the sentiment, emotion, a summary 
and a score out of 10 — all neatly displayed!

## Example

Input: "I love coding but sometimes it frustrates me"

Output:
- Sentiment: mixed
- Emotion: happy  
- Summary: Speaker enjoys coding but finds it challenging
- Score: 7/10

## How to run it

1. Install the library: pip install google-genai
2. Add your Gemini API key
3. Run: python sentiment_analyzer.py

## Built with

Python 3.14 | Google Gemini API | JSON parsing | Prompt Engineering
